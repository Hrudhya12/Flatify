# Flatify

Flatify is a Django web app that helps flatmates rotate household tasks according to their chances and the rotation pattern.  
It assigns tasks, tracks completion, and makes sure no one gets the same task twice in a row. This also helps to complete the tasks on time.

# Features
- Task rotation
- Tracking the completion of tasks
- Login
- Recent task history
- Automated tests
- GitHub Actions CI

# How to Run the Project
1. Clone the repository  
2. Create and activate a virtual environment  
3. Install dependencies  
4. Apply migrations  
5. Create a superuser  
6. Start the server
 
# bash
python manage.py runserver

# Running tests
python manage.py test
